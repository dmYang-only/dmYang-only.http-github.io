ASCII表是Unicode字符集的子集

byte,short,char这几个数据类型一旦参与任何形式的运算，都会先将这个数据类型自动的提升到int类型在去赋值

一、元字符

​ \n换行符 \t制表符(8个字符) \ "转义字符

​ \x 表示十六进制数 \ 表示八进制数

​ \u后面加上四个十六进制代码来表示Unicode字符,long a=’\u00FF’

​ \b 退格 ，\f换页 ，\r回车

二、标识符

由26个英语字母a-Z，0-9，$, _组成，可以为中文

  • 不能以数字开头

  • 不能使用关键字和保留关键字（go ，const常量）

  • this_is_Ok可以是标识符

三、关键字

static 、void、short、interface、final

四、常量

​ 空常量：null

五、变量

格式:赋值类型 变量名=常量值; 声明–>赋值–>使用

​ 就是给类或者接口等取名字时候用的字符序列

  • 一个变量只能在同一作用域作用，同一作用域不能出同名变量，int a=10;
  • 类成员变量、局部变量(在作用域里面才能用)

六、数据类型

1、String

字符串是一个类，

属于引用数据类型，String str=new String(“好的”);

2、整数类型

3、类型转换

  • 自动类型转换：从小到大

  • 强制类型转换：从大的数据到小的数据，boolean类型、String不参与任何形式转换

  • 二进制数前面加0b，十六进制加0x，八进制加0
  • 根据ASCII编码，48–>0 65–>A 97–>a

误区：类体只能有声明和方法，不能存在语句；

七、运算符

单目运算符〉算术运算符〉比较运算符〉逻辑运算符（除了“非”运算）

byte,short,char这几个数据类型一旦参与任何形式的运算，都会先将这个数据类型自动的提升到int类型在去赋值

1、算术运算符

2、赋值运算符

赋值左边不能为表达式。a=b+3=c错误;

扩展运算符本身就隐藏一个强制类型转换的问题

3、比较运算符

结果都是boolean型 == ， ！=，>=

4、逻辑运算符

逻辑与& 逻辑或|，逻辑非 ！，^逻辑异或

&&具备短路效果，即左边的数据如果为false，那么不在执行右边的数据

||短路或

5、位运算符

位运算符，转化为二进制补码比较, 最高位为1要取反+1

按位与&，按位或|，按位取反~，按位异或^

<<左移（右边空位补0），>>右移

6、三目运算符

int max=a>b? a:b;

String height=a>b? “a大”:”b大”
